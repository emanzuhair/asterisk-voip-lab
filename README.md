# asterisk-voip-lab
My first Asterisk VoIP lab built using Ubuntu, PJSIP, Zoiper and MicroSIP.
This is my first hands-on Asterisk project.

I built this lab to better understand how SIP registration and basic VoIP calls work.

## What I used

- Hyper-V - I always prefer it more than VMware and others VM software :)
- Ubuntu 22.04
- Asterisk 22
- PJSIP
- Zoiper
- MicroSIP


## What I learned

- Installing Asterisk on Ubuntu
- Configuring PJSIP endpoints
- Creating SIP extensions
- Registering softphones
- Making extension-to-extension calls
- Using Asterisk CLI for troubleshooting
## Challenges I faced

This project didn't work on the first try.

Some of the issues I had to troubleshoot were:

- Missing PJSIP transport
- SIP registration timeout (408)
- Registering both endpoints successfully

Working through these issues helped me understand Asterisk much better.

## Next Steps

I plan to continue improving this lab by adding more Asterisk features and learning more about VoIP.
